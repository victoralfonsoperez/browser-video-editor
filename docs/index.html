Features

🔒

100% Private

All processing happens locally with FFmpeg WebAssembly. Your videos are never uploaded anywhere.

🎬

Multiple Formats

Export as MP4, WebM, MOV, or GIF with configurable quality and resolution settings.

✂

Multi-Clip Workflow

Create multiple named clips from a single video, reorder them, and batch export with a queue.

⌨

Keyboard Shortcuts

Professional editing shortcuts — J/K/L for playback, I/O for in/out points, comma/period for frame stepping.

🖥

Free & Open Source

No sign-up, no watermarks, no limits. Open source on GitHub under a permissive license.

📱

Works Everywhere

Responsive design for desktop, tablet, and mobile with touch-optimized interactions.
